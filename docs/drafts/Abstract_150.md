# Abstract (150 words)

Small nonprofits delegate bookkeeping while financially responsible persons (FRPs) cannot monitor ledgers continuously between audits; asset misappropriation by a trusted bookkeeper persists undetected, and episodic controls induce warning fatigue.

Next-Generation Accounting Monitoring Engine (NGAME) is an open-source, decision science-based design science artifact designed, demonstrated, and evaluated following Design Science Research Methodology (DSRM). The pipeline produces daily Resource Description Framework (RDF)/Web Ontology Language (OWL) ledger snapshots, a Churn Persistence Index (CPI) behavioral fingerprint, and interpretable management warnings for the FRP, with locally hosted large language model (LLM) misappropriation triage.

Evaluation under DSRM Activities 4–5 combines sandbox validation, agent-driven anomaly injection, CPI metric analysis, and observational field deployment in consenting small nonprofits using QuickBooks Online (QBO).

Contributions are: (1) a readily implemented continuous forensic surveillance tool whose reporting is meaningful to non-expert FRPs; (2) churn as an innovative daily behavioral fingerprint; and (3) prioritized evidence of anomalous activity for FRP final adjudication.
